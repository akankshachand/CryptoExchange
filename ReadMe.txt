This webpage dispalys the prices of two cryptocurrencies (Bitcoin and Ethereum) from Bitstamp and Alpha Vantage. It also recommends the user on which exchange one should buy from and/or sell to.

Make sure Node.js and the following dependencies installed:
1. npm install express
2. npm install express-handlebars
3. npm install request
4. npm install node-fetch --save
5. npm install -D nodemon [Optional]

To run the application enter go to app directory in the command prompt and enter:
node index.js
OR
npm run dev [if you have nodemon installed]
